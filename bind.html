Bind Functionality in Javascript (Bind, Call, Apply)

Here, you can see the simple example that describes the working of bind, call functions.

  • Bind - updates the reference
  • Call - updates and also let's you pass arguments as comma seperated variables
  • Apply - updates and also let's you pass arguments as array

This updates the original reference with the reference supplied.

For ex. By default, the reference to 'this' will always be the reference that invoked a particular function. If a function is called on global scope, the reference usually is 'window'. If a function is called on click of any event, the reference usually is the Event object

You can go ahead and change the reference by using the following approaches

	//Code
	function getDetails (name, age, location) {
		console.log(this.name, this.age, this.location );
	}
	
	var arun = {
		name: 'Arunkumar',
		age: 25,
		location: 'India'
	};
	
	var kumar = {
		name: 'Kumar',
		age: 26,
		location: 'Somewhere in India'
	};
	
	getDetails.call(arun); //Returns Arunkumar 25 India
	getDetails.call(kumar); //Returns Kumar 26 Somewhere in India
	
	getDetails.bind(arun).call(); //Returns Arunkumar 25 India
	getDetails.bind(kumar).call(); //Returns Kumar 26 Somewhere in India
	
	getDetails(); //Returns undefined undefined and location of the window(object)
